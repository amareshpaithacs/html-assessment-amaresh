HTML Tag Reference

This page is a quick reference for the most commonly used HTML tags. I made this reference sheet while studying for this assessment.

Structure Tags

Tag Full Name Purpose
<html> HTML root element Wraps the entire HTML document
<head> Document head Contains meta information and links
<body> Document body Contains all visible page content
<header> Page header Introductory content or navigation
<nav> Navigation Contains navigation links
<main> Main content The main unique content of the page
<section> Section A thematic grouping of content
<article> Article Self-contained piece of content
<aside> Aside / Sidebar Content related to but aside from main
<footer> Page footer Footer with contact, copyright info

Text Formatting Tags

Tag Effect Example Output
<b> Bold text (presentational) Bold text
<strong> Strong importance (semantic bold) Strong text
<i> Italic text (presentational) Italic text
<em> Emphasis (semantic italic) Emphasized text
<u> Underlined text Underlined
<s> Strikethrough text Strikethrough
<mark> Highlighted text Highlighted
<code> Inline code print("hello")
<sub> Subscript H2O
<sup> Superscript x2
