Example 1 - Basic HTML Page Structure

This is the basic structure I used for every page in the assessment:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Name | Amaresh</title>
    <link rel="icon" href="images/favicon.ico">
</head>
<body bgcolor="#fafafa">
    <header>...</header>
    <nav>...</nav>
    <main>...</main>
    <footer>...</footer>
</body>
</html>

Example 3 - HTML Form with Validation

This example shows basic form elements from Task 4:

<form action="#" method="post">

    <fieldset>
        <legend>Personal Details</legend>

        <label for="name">Full Name:</label>
        <input type="text"
               id="name"
               name="name"
               required
               minlength="3"
               placeholder="Enter your name">

        <label for="email">Email:</label>
        <input type="email"
               id="email"
               name="email"
               required>

    </fieldset>

    <input type="submit" value="Submit">
    <input type="reset" value="Clear">

</form>
